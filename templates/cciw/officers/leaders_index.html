Summary help for leaders

For leaders, or those helping leaders with admin tasks, you have a number of responsibilities that need to be attended to:

  • First, you need to create your officer list
  • Once you have created this, you should email your team (using the addresses provided) to ask them to fill in their application forms.
  • Officer lists and application forms should be done as early as possible, so that:
    • You can start collecting references
    • We can ensure that we have valid DBS checks done before camp
    • We can reserve places for children of officers who are coming on camp
  • Ensure that your officers with children on camp have completed their bookings. This must be done using the normal bookings system and in time for the end of the initial booking period.
  • Once the initial booking period is done, you will need to use the booking queue page to add some extra information (such as indicating which places are for the children of officers). This must be done as soon as possible to allow the booking secretary time to finish this process.
  • Once applications start to be completed, you can request references for each officer.
  • Near to camp, you will need to download spreadsheets containing officer and camper data
